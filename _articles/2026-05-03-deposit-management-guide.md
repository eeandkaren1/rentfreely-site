<h3>押金的正確認知</h3>
<p>很多房東把押金視為「最後一個月的保障」或「破壞賠償的保險」，這個認知是正確的——但前提是扣款要有依據。押金不是房東的收入，是代為保管的擔保金，扣款必須說得清楚，剩餘金額必須返還。</p>
<h3>押金金額：收多少最恰當？</h3>
<p>上限是兩個月租金，但收多少由你決定：</p>
<ul>
<li><strong>一個月押金：</strong>對房客負擔較輕，吸引力較高，但保障相對薄</li>
<li><strong>兩個月押金：</strong>最常見的選擇，兼顧保障與吸引力</li>
<li><strong>按狀況調整：</strong>允許養寵物、分租等高風險情況，建議收滿兩個月</li>
</ul>
<h3>押金收據的正確開立方式</h3>
<p>收取押金時必須開立收據，收據應載明：</p>
<ul>
<li>房屋地址</li>
<li>押金金額（數字及中文）</li>
<li>收取日期</li>
<li>出租人（房東）姓名與簽章</li>
</ul>
<h3>退租時押金扣款的合法依據</h3>
<p>可以從押金扣除的項目：</p>
<ul>
<li>欠繳租金（有明確金額）</li>
<li>可舉證的損壞修繕費（需有估價單和損壞照片）</li>
<li>合約明定的清潔費（如約定退租需專業清潔）</li>
<li>合約明定的違約金（如提前解約）</li>
</ul>
<p>不可扣除的項目：</p>
<ul>
<li>自然耗損的修繕費用</li>
<li>沒有依據的模糊費用（「整理費」「管理費」等）</li>
<li>入住前就已存在的損壞</li>
</ul>
<h3>最常見的押金敗訴原因</h3>
<p>房東在押金糾紛中最常敗訴的情況：</p>
<ul>
<li>沒有入住點交記錄，無法證明損壞是房客造成</li>
<li>以油漆費、清潔費等自然耗損項目扣款</li>
<li>扣款沒有單據（沒有估價單或收據）</li>
<li>全額沒收押金，遠超過實際損害</li>
</ul>
<h3>退租押金結算的正確程序</h3>
<ol>
<li>點交當天雙方確認屋況，有損壞立即拍照</li>
<li>取得損壞修繕的估價單（至少兩家估價）</li>
<li>書面列出扣款明細，逐項說明</li>
<li>押金扣除後，剩餘金額於合理期間內（通常兩週）返還</li>
<li>保留所有扣款單據至少五年</li>
</ol>
