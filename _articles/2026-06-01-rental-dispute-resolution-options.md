<h3>糾紛不等於撕破臉</h3>
<p>很多人遇到租屋糾紛，要不是忍著不講、要不是直接開戰。其實在這兩個極端之間，有很多效率高、成本低的解決管道。選對管道，才能以最小的成本拿回應有的權利。</p>
<h3>五種解決管道的比較</h3>
<ul>
<li><strong>直接協商：</strong>最快、零費用。書面提出訴求，態度理性，先給對方解決機會。重點：全程書面留存。</li>
<li><strong>鄉鎮市區調解委員會：</strong>免費、1至2個月、效力同確定判決。九成租屋糾紛的最佳解。向任一方戶籍地或房屋所在地調解委員會申請。</li>
<li><strong>縣市地政局租賃爭議調處：</strong>免費、專業性高，適合涉及定型化契約的複雜爭議。向房屋所在縣市地政局申請。</li>
<li><strong>小額訴訟：</strong>適用10萬元以下請求，費用低廉（按請求金額約1%），程序簡便，可自行提起，不需律師。</li>
<li><strong>一般民事訴訟：</strong>適合大金額或複雜案件，但耗時（6個月至一年以上）且律師費可觀，作為最後手段。</li>
</ul>
<h3>根據糾紛類型選擇管道</h3>
<ul>
<li><strong>押金糾紛（3至10萬以下）：</strong>調解委員會 → 若不成，小額訴訟</li>
<li><strong>電費超收（金額較小）：</strong>直接協商 → 調解委員會</li>
<li><strong>修繕費用（爭議較複雜）：</strong>地政局調處 → 調解委員會</li>
<li><strong>要求遷讓房屋（房東驅逐糾紛）：</strong>調解委員會 → 確認自己的合法居住權</li>
</ul>
<h3>申請調解的準備工作</h3>
<ul>
<li>整理爭議事實的時間軸（何時發生、何時通知、對方如何回應）</li>
<li>整理相關文件：租約、繳費記錄、通訊截圖、照片</li>
<li>明確的請求金額（我要求返還押金XX元、電費差額XX元）</li>
<li>填寫聲請書（調解委員會提供格式）</li>
</ul>
<h3>調解時的注意事項</h3>
<ul>
<li>態度理性，陳述事實為主</li>
<li>有彈性但有底線：設定不可退讓的最低接受條件</li>
<li>達成協議後，要求在調解書上載明所有細節（金額、期限、方式）</li>
<li>調解書送請法院核定後，效力等同確定判決</li>
</ul>
