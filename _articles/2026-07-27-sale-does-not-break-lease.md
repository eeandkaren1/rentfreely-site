<h3>房子換了主人，租約會怎樣</h3>
<p>租到一半，收到通知說房子賣掉了，新屋主要你搬走。這是租屋族最沒有安全感的情境之一。</p>
<p>《民法》第425條第1項對此有明確規定：<strong>「出租人於租賃物交付後，承租人占有中，縱將其所有權讓與第三人，其租賃契約，對於受讓人仍繼續存在。」</strong>這就是俗稱的「買賣不破租賃」。</p>
<p>簡單說：<strong>房子賣了，租約跟著走。</strong>新屋主承受原房東在租賃契約上的地位，租約繼續有效，房客不必因為換屋主就搬家。</p>

<h3>原則的適用要件</h3>
<p>依條文文義，「買賣不破租賃」的保護須具備兩個前提：</p>
<ul>
<li><strong>租賃物已經交付</strong>——房東已把房子交給房客</li>
<li><strong>承租人占有中</strong>——房客實際上正在占有使用該房屋</li>
</ul>
<p>換句話說，保護的是<strong>已經實際入住的房客</strong>。若只簽了約還沒交屋，情況可能有所不同。</p>
<p>此外，本條的適用不限於「買賣」。條文用的是「將其所有權讓與第三人」，因此贈與、繼承等所有權移轉的情形，原則上也在射程範圍內。</p>

<h3>關鍵例外：五年以上且未經公證</h3>
<p>這是整條規定中最需要注意的部分。《民法》第425條<strong>第2項</strong>規定：</p>
<p><strong>「前項規定，於未經公證之不動產租賃契約，其期限逾五年或未定期限者，不適用之。」</strong></p>
<p>拆解這個例外的要件：</p>
<ul>
<li><strong>未經公證</strong>，且</li>
<li><strong>期限超過五年，或未定期限（不定期租賃）</strong></li>
</ul>
<p>兩個條件同時成立時，買賣不破租賃的保護<strong>不適用</strong>。也就是說，房子易主後，新屋主可能不受原租約拘束。</p>
<p>反過來理解，這個例外告訴我們兩件事：</p>
<ol>
<li><strong>五年以內的定期租約，即使沒有公證，仍受保護。</strong>絕大多數的租屋（一年、兩年約）都落在這個範圍，房客不必過度擔心。</li>
<li><strong>長期租約（超過五年）或不定期租約，公證就變得關鍵。</strong>經過公證，即使期限逾五年，仍可受買賣不破租賃保護。</li>
</ol>
<p>這是公證在事前風險管理上的另一項價值——不只是為了取得執行名義，也關係到租賃關係能否對抗新的所有權人。</p>

<h3>對房客的實務意義</h3>
<p><strong>一般情形（一至兩年租約）：</strong>房子易主後，租約原則上繼續有效。新屋主承受原房東的地位，房客繼續依原條件履行，向新屋主繳租。</p>
<p><strong>需要注意的實務問題：</strong></p>
<ul>
<li><strong>押金怎麼辦：</strong>押金的返還責任在實務上是常見爭點。房客應保留原始的押金收據與匯款紀錄，並在得知易主時，向新舊屋主確認押金的移轉與返還責任歸屬，最好留下書面。</li>
<li><strong>租金付給誰：</strong>應確認所有權移轉的時點與新屋主身分（可調閱建物謄本核對），避免付錯對象而衍生爭議。</li>
<li><strong>新屋主要求重簽約：</strong>房客沒有義務接受不利於己的新條件；但若雙方合意重新簽約，原契約條款即可能被取代，簽署前應審慎比對。</li>
<li><strong>租約期限逾五年：</strong>若你的租約是長期約，建議確認是否已辦理公證。</li>
</ul>

<h3>對房東的實務意義</h3>
<p><strong>出售方（原房東）：</strong>房屋出售時，若屋內有租約存在，這是必須向買方揭露的重要事項。隱匿租約存在可能衍生買賣契約上的爭議。實務上，帶租約出售的房屋，其價格與買方族群本來就與空屋不同。</p>
<p><strong>買受方（新屋主）：</strong>購買前務必確認標的物是否有租約、租期到什麼時候、押金多少、由誰返還。這些事項應在買賣契約中明確約定。若買方是為了自住而購屋，卻買到帶有長期租約的房屋，後續處理會相當棘手。</p>

<h3>常見的三個誤解</h3>
<ul>
<li><strong>「房子賣了租約就自動失效」：</strong>錯誤。這正是民法425條要防止的情況。</li>
<li><strong>「新屋主說要收回自住，我就必須搬」：</strong>不必然。租賃關係若依425條繼續存在，新屋主仍應受租約拘束；能否終止須依契約與法律規定判斷。</li>
<li><strong>「有沒有公證都一樣」：</strong>對五年以內的定期租約影響不大，但對逾五年或不定期租約，公證與否可能是保護有無的分水嶺。</li>
</ul>

<h3>建議的自保動作</h3>
<ul>
<li><strong>保存租約正本</strong>以及所有租金、押金的支付憑證</li>
<li><strong>保留交屋與入住紀錄</strong>，證明「租賃物已交付且占有中」</li>
<li>得知房屋易主時，<strong>調閱建物登記謄本</strong>確認新所有權人，並以書面（LINE、電子郵件、存證信函皆可）向新舊屋主確認租約承受與押金歸屬</li>
<li>租期較長者，評估辦理<strong>公證</strong>的必要性</li>
</ul>
