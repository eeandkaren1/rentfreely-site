<h3>退租比入住更需要準備</h3>
<p>很多人把心力放在入住前，卻沒有好好準備退租。但其實退租才是最容易發生糾紛的環節——押金能不能全額拿回，完全取決於退租這幾週的準備是否充分。</p>
<h3>退租倒數時程表</h3>
<p><strong>D-30（退租前30天）：</strong></p>
<ul>
<li>依合約規定時限，書面通知房東退租日期（LINE截圖存檔）</li>
<li>確認押金返還方式（匯款帳號、還款期限）</li>
<li>開始確認屋況，評估是否有需要修補的地方</li>
</ul>
<p><strong>D-14（退租前14天）：</strong></p>
<ul>
<li>預約專業清潔（如需要）</li>
<li>開始搬運物品，不要等到最後一天</li>
<li>大型廢棄物預約免費清運</li>
</ul>
<p><strong>D-7（退租前7天）：</strong></p>
<ul>
<li>進行基礎清潔（不是恢復如新，而是基本可使用狀態）</li>
<li>確認點交時間（與房東約定）</li>
<li>整理所有鑰匙、磁扣、遙控器</li>
</ul>
<p><strong>D-Day（點交當天）：</strong></p>
<ul>
<li>全程錄影點交</li>
<li>抄錄水電表讀數（拍照）</li>
<li>交還所有鑰匙</li>
<li>確認繳清水電費和租金</li>
<li>雙方簽署點交單</li>
</ul>
<p><strong>D+7（退租後7天）：</strong></p>
<ul>
<li>辦理戶籍遷出（如已設籍）</li>
<li>更改各類帳單寄送地址</li>
<li>確認押金入帳</li>
</ul>
<h3>退租清潔的合理標準</h3>
<p>退租清潔不是要讓房子恢復到「全新」狀態，而是「基本可使用的清潔狀態」：</p>
<ul>
<li>地板清掃、拖地</li>
<li>廚房清潔（爐台、油煙機、水槽）</li>
<li>浴室清潔（馬桶、洗手台、浴室）</li>
<li>垃圾清除</li>
<li>個人物品全數帶走</li>
</ul>
<p>自然耗損（油漆褪色、輕微刮痕）不需要由你修復，更不能成為扣押金的依據。</p>
<h3>如何對應退租時的扣款要求</h3>
<p>房東提出扣款時：</p>
<ol>
<li>要求房東列出完整的扣款明細（每項說明+估價單）</li>
<li>對照入住時的照片影片，判斷哪些損壞是入住前已有或自然耗損</li>
<li>有理據的扣款可接受，無依據的扣款書面提出異議</li>
<li>協商不成，申請調解</li>
</ol>
