<h3>好房客的重要性</h3>
<p>「房子好不好租不重要，重要的是租給對的人。」這句話對有出租經驗的房東來說，是深刻的體會。一個好房客，按時繳租、愛惜房屋、溝通順暢，讓你幾年都不需要操心。一個問題房客，可能讓你損失幾個月的租金、付出數倍的管理時間，甚至引發法律糾紛。</p>
<h3>合法且合理的查驗項目</h3>
<ul>
<li><strong>身分核對：</strong>要求本人攜帶身分證，當面確認照片與本人一致</li>
<li><strong>工作收入：</strong>了解職業性質，確認有穩定收入。月租金建議不超過月收入的三分之一</li>
<li><strong>緊急聯絡人：</strong>留存姓名與電話，作為必要時的聯繫管道</li>
<li><strong>租賃記錄查詢：</strong>使用本站裁判書查詢助手，搜尋有無租賃相關訴訟記錄（欠租、遷讓等）</li>
</ul>
<h3>面談時的觀察重點</h3>
<ul>
<li><strong>對屋況的關注度：</strong>認真詢問設備狀況、管線細節的房客，通常是真的想長租、想好好住的人</li>
<li><strong>對合約的態度：</strong>願意仔細看合約並提問，代表他理性、謹慎；不看就簽，反而要小心</li>
<li><strong>準時：</strong>看房準時到達是一個小細節，但反映時間觀念</li>
<li><strong>溝通方式：</strong>禮貌、清楚、理性，是良好租賃關係的基礎</li>
</ul>
<h3>不能作為拒租理由的因素（歧視性拒租）</h3>
<p>以下因素不能作為拒租的依據，否則可能涉及歧視：</p>
<ul>
<li>性別（包含性別認同）</li>
<li>婚姻狀況（已婚/未婚/離婚）</li>
<li>有無子女</li>
<li>國籍或種族</li>
<li>宗教信仰</li>
<li>身心障礙狀況</li>
</ul>
<h3>其他注意事項</h3>
<ul>
<li>不得要求提供存摺明細、薪資單等過度個資（只需確認有穩定收入）</li>
<li>不得以性別、年齡等理由設定限制性廣告（如「限女性」「限年輕人」）</li>
<li>同等條件的多位申請者，以先來先得為原則，不能以歧視性理由挑選</li>
</ul>
