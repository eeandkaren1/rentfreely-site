<h3>為什麼要談這個題目</h3>
<p>房客積欠三個月租金、聯絡不上、也不搬走。房東的損失每天都在累積，而正式的法律程序動輒數月。在這種壓力下，「乾脆把電停掉」「把鎖換掉」成為許多房東腦中浮現的第一個念頭。</p>
<p>這篇文章不是要指責誰，而是要說明一件事：<strong>這類做法在我國法制下屬於「自力救濟」，而自力救濟的容許範圍相當狹窄。</strong>了解這個框架，對房東和房客都有意義。</p>

<h3>制度的基本立場：權利實現要透過程序</h3>
<p>我國民事法制的基本設計是：權利受侵害時，應循國家提供的程序（調解、訴訟、強制執行）實現權利，而非由當事人自行以實力排除。</p>
<p>《民法》第151條雖然規定了自助行為，但要件嚴格——須「為保護自己權利，對於他人之自由或財產施以拘束、押收或毀損者，不負損害賠償之責。但以不及受法院或其他有關機關援助，並非於其時為之，則請求權不得實行或其實行顯有困難者為限」。同法第152條並要求行為人須即時向法院聲請處理。</p>
<p>換言之，自助行為是為了處理「來不及找法院、不做就來不及」的緊急情況，而非用來替代正常的租賃糾紛處理程序。</p>

<h3>相關的法律評價脈絡</h3>
<p>斷水斷電、換鎖等行為，在實務上可能被放在以下幾個規範框架下檢視：</p>
<p><strong>民事層面：</strong></p>
<ul>
<li>出租人依《民法》第423條負有「以合於所約定使用收益之狀態」交付並維持租賃物的義務。斷水斷電是否違反此項義務，是常見的爭點。</li>
<li>因此造成承租人損害者，可能涉及債務不履行的損害賠償問題。</li>
</ul>
<p><strong>刑事層面：</strong></p>
<ul>
<li>《刑法》第304條強制罪規定：「以強暴、脅迫使人行無義務之事或妨害人行使權利者」。實務上，以斷絕水電或變更門鎖之方式，使他人無法正常使用其居住空間，是否構成本罪的「妨害人行使權利」，是司法實務上曾被討論的類型。</li>
<li>若過程中損壞他人物品，另可能涉及《刑法》第354條毀損罪；擅自處分屋內財物，則可能涉及侵占相關規定。</li>
</ul>
<p><strong>需要強調的是：</strong>個別案件是否成立犯罪或損害賠償責任，取決於具體事實——包括行為態樣、當事人間的契約約定、租賃關係是否已合法終止、行為當時的情境等，法院見解亦可能因案而異。本文無法、也不會對任何特定情形下結論。</p>

<h3>對房東的實務風險提醒</h3>
<p>從風險管理角度看，自力救濟對房東的問題在於<strong>風險與收益不成比例</strong>：</p>
<ul>
<li>省下的是幾個月的程序時間</li>
<li>換來的可能是民事賠償責任、刑事偵查程序，以及在後續租賃糾紛中的不利地位</li>
<li>更現實的是——即使斷了水電，房客仍然占有房屋，房東還是收不回房子，問題並未解決</li>
</ul>
<p>相對地，正規途徑雖然慢，但終點是明確的：取得執行名義後聲請強制執行，由執行人員依法處理遷讓與屋內物品，房東的地位受到程序保障。</p>

<h3>房東可以循的正規途徑</h3>
<p>依租賃關係的狀態，一般而言可考慮的方向包括：</p>
<ol>
<li><strong>依契約及法律規定合法終止租約</strong>——租金積欠達一定程度時，民法及土地法對出租人的終止權有相關規定，但通常須先經催告程序</li>
<li><strong>寄發存證信函</strong>，正式催告並固定時間點</li>
<li><strong>申請鄉鎮市區調解</strong>（免費，成立後經核定具執行名義效力）</li>
<li><strong>依金額選擇支付命令或訴訟</strong>，取得執行名義</li>
<li><strong>聲請強制執行</strong>，由法院處理遷讓與屋內物品</li>
</ol>
<p>若租約當初有辦理<strong>逕受強制執行公證</strong>，前面幾步可能得以大幅簡化——這正是公證在事前值得投資的原因。</p>

<h3>對房客的實務提醒</h3>
<p>若遇到水電遭中斷或門鎖被更換的情形，實務上常見的處理方向包括：</p>
<ul>
<li><strong>立即保存證據</strong>：拍照、錄影（含可辨識的時間資訊）、保留電費單、與房東的對話紀錄</li>
<li><strong>以可留存的方式與房東聯繫</strong>，記錄對方的說法</li>
<li>視情況<strong>報警處理</strong>並索取報案三聯單，警方到場紀錄本身即為證據</li>
<li><strong>尋求專業協助</strong>：法律扶助基金會、縣市政府免費法律諮詢</li>
</ul>
<p>同時也要誠實面對：如果自己確實有欠租，這並不會因為房東的處理方式不當而消失。兩件事在法律上是分開評價的，實務上也常見雙方各有主張的情形。</p>

<h3>回到根本：預防比處理便宜</h3>
<p>會走到自力救濟這一步的租賃關係，通常在更早的階段就已經出現問題。事前可以做的包括：</p>
<ul>
<li>簽約前的<strong>房客條件確認</strong>，以及書面契約條款的完整約定</li>
<li>租約辦理<strong>公證</strong>，特別是逕受強制執行公證</li>
<li>租金遲繳時<strong>及早以書面催告</strong>，不要拖到累積數月</li>
<li>保留完整的<strong>金流與溝通紀錄</strong></li>
</ul>
