<h3>維護vs急修：數字告訴你答案</h3>
<p>一台使用5年的冷氣，每半年清洗濾網（費用約500至800元），正常可以再用5至8年。若長期不清洗，壓縮機可能在7至8年就損壞，更換費用約15,000至25,000元。一次急修的費用，相當於15至25次的定期保養費。</p>
<h3>年度維護時程表</h3>
<p><strong>每半年（建議春季和秋季）：</strong></p>
<ul>
<li>冷氣濾網清洗（房客可自行清洗，但房東定期確認更佳）</li>
<li>排水孔清潔（浴室、廚房、陽台）</li>
<li>門鉸鏈、窗戶軌道上油</li>
<li>確認廚房、浴室矽利康無裂縫或霉斑</li>
</ul>
<p><strong>每年（建議梅雨季前）：</strong></p>
<ul>
<li>屋頂防水狀況視察（有空屋頂者）</li>
<li>外牆、窗框密封膠確認</li>
<li>熱水器安全閥測試、燃燒效率確認</li>
<li>住宅用火災警報器電池更換（或測試）</li>
<li>電路跳電保護開關測試</li>
</ul>
<p><strong>每兩至三年：</strong></p>
<ul>
<li>全室矽利康更新（廚房、浴室、窗邊）</li>
<li>水龍頭、蓮蓬頭墊圈和濾網更換</li>
<li>冷氣室外機清洗（深度保養）</li>
<li>排汙管路專業疏通</li>
</ul>
<h3>維護記錄的重要性</h3>
<p>每次維護後留存記錄（照片、廠商單據），這份記錄的用處：</p>
<ul>
<li>日後若發生設備糾紛，可證明有善盡保養義務</li>
<li>了解設備使用年限，提前規劃更換預算</li>
<li>向房客說明保養結果，建立信任關係</li>
</ul>
<h3>如何安排維護事宜不打擾房客？</h3>
<ul>
<li>固定在每半年的固定時段（如春節後、中秋後）統一排程</li>
<li>提前一週以上通知房客，確認時間</li>
<li>維護時間控制在合理範圍（不超過兩至三小時）</li>
<li>維護後告知完成情況，讓房客安心</li>
</ul>
<h3>主動維護對出租收益的影響</h3>
<p>設備維護良好的物件，房客滿意度高，續租率高，空置期短。以避免一次兩個月的空置期為例（2萬月租），省下的40,000元，相當於80次的冷氣清洗費。良好的維護，是隱形的租金優化。</p>
